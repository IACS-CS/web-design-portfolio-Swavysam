The project aims to showcase proficiency in translating print design into a web-friendly format while emphasizing precision in visual details.

THINGS I LEARNED

Java Script: JavaScript is a high-level, interpreted programming language designed primarily for client-side web development. It enables developers to provide dynamic and interactive content for web browsers. JavaScript, sometimes known as JS, is a fundamental technology for developing modern web applications.

HTML: HTML, or HyperText Markup Language, is the standard markup language used to define the structure of web pages. It includes a collection of elements or tags that specify the various components of a web page, such as headings, paragraphs, links, images, forms, and so on. HTML is a critical component of web development, serving as the foundation for material on the World Wide Web.

CSS: CSS, or Cascading Style Sheets, is a style sheet language that describes how an HTML or XML page should be presented. CSS controls the layout, look, and formatting of items on a web page. It isolates a document's structure and content (which HTML handles) from its visual presentation, making it easier to maintain and apply uniform styling across different pages.
